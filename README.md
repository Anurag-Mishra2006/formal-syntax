
# Formal Syntax 

> **"Syntax is the structure; Formal is the rigor."**

This is the source code for my personal engineering log, [formalsyntax.dev](https://github.com/Anurag-Mishra2006/formal-syntax). This site serves as a public ledger of my journey through web architecture, mathematical logic, and systems engineering.

Inspired by the pragmatic philosophy of Linus Torvalds, this project focuses on high-performance, minimalist design, and technical depth.

## 🧠 Core Philosophy
- **Mathematical Rigor:** Every piece of software has an underlying logic. I explore the discrete math behind the code.
- **Systems Thinking:** Moving beyond surface-level frameworks to understand how the "metal" and the "browser" actually work.
- **Impact-Driven:** Documentation is the first step toward building tools that create systemic change.

## 🛠 Tech Stack
- **Engine:** [Hugo (Extended)](https://gohugo.io/) - v0.154.5
- **Theme:** [PaperMod](https://github.com/adityatelange/hugo-PaperMod) (Customized)
- **Math Rendering:** [KaTeX](https://katex.org/) (Fast LaTeX support)
- **Search:** [Pagefind](https://pagefind.app/) (Static search)
- **Hosting:** [Cloudflare Pages](https://pages.cloudflare.com/)

## 🚀 Local Development

To run this project locally, you need the Hugo Extended version.

1. **Clone the repository with submodules:**
   ```bash
   git clone --recursive https://github.com/Anurag-Mishra2006/formal-syntax.git
   cd formal-syntax
   ```

2. **Start the development server:**
   ```bash
   hugo server -D
   ```

3. **Build for production:**
   ```bash
   hugo
   ```
